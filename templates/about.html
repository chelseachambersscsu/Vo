About MemoBoat

MemoBoat is a meeting memo generator that transforms messy, raw meeting notes into clear, structured memos in seconds. Simply paste your notes, choose the meeting type, and let AI distill the conversation into a summary, action items, and key decisions.

All memos are stored in a local SQLite database so you can revisit them at any time. You can also download any memo as a plain-text file for easy sharing.

Tech Stack
HTML5 CSS3 Bootstrap 5 Vanilla JavaScript Flask + Jinja2 SQLite3 OpenAI API
Database Schema

meeting_types (type_id PK, type_name)

transcripts (transcript_id PK, type_id FK, raw_notes, timestamp)

summaries (summary_id PK, transcript_id FK, summary, action_items, key_decisions)
